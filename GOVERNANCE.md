# Governance

Repositories under this account are maintained to support practical GIS, data, and forestry technology work.

## Maintainer Responsibilities

Maintainers are responsible for:

- Keeping repository purpose and scope clear.
- Reviewing issues and pull requests.
- Protecting sensitive data and credentials.
- Keeping automation, documentation, and dependencies maintainable.
- Deciding when repository-specific guidance should override these defaults.

## Project Decisions

Project decisions should prioritize:

- Public service value.
- Data quality and reproducibility.
- Security and privacy.
- Clear documentation.
- Sustainable maintenance.

## Repository Ownership

Each repository may define its own owners, reviewers, and release practices. When a repository does not define its own process, these default expectations apply.
